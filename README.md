# FinSight AI

AI-powered global financial intelligence platform — stock forecasting,
global news sentiment, RAG over SEC filings, MCP server, and an autonomous
analyst agent. Built fresh, phase by phase.

## Status
Phase 1 — setup complete. Ingestion, dbt pipeline, ML, AI layers, and
dashboard to follow.

## Stack
Python · DuckDB · Parquet · dbt · Prefect · PostgreSQL · Qdrant ·
Prophet · FinBERT · Isolation Forest · MLflow · LangChain/LangGraph ·
MCP · FastAPI · Streamlit · Docker
